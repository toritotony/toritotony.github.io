Computer Science & Fusion Development & Software Engineering


RAG Financial Analyst & Report Generator

This web application is hosted by Streamlit and is responsible for providing retail investors with quick information on assets, stocks, sectors, and compares assets to help individuals make informed decisions


Web Application for multimedia RAG

Python driven Flask application that allows the user to submit different forms of media (file, link-to-file, link-to-video) to "ask it" questions. Uses OpenAI's GPT 3.5 Turbo Model and was temporarily deployed with CI/CD through Github Actions and Azure Web Apps.


OpenAI Voice Activated Assistant

This Flask-based web application provides users the ability to input a voice recording through the device's microphone, and receive an audible output that reads a response given by OpenAI. Think of it as a verbal conversation with ChatGPT instead of textual.


Power Platform Solution for Citywide 2030 Strategic Vision Plan

This low code solution leveraged Power Apps, Power Automate, and SharePoint Lists to allow department employees at the City of Long Beach to submit records relating to action items, accomplishments and notes for the City's 2030 Strategic Vision.


Azure Web App Auto-Deployment via Github Actions

Utilized Azure App Service and Web Apps, along with GitHub Actions to auto deploy flask-based web applications from development to production for every branch commit. It delegates authority to a service principal to execute this, and relies on stored secrets and API keys to run the required logic.


Multistage Approval Flows w/ Additional Recurring Flows

Provided city managers with the ability to review grants submitted through Microsoft Forms while approving the grant request and assigning it to the next individual required to accept or reject. Can also restart approval through button in SharePoint list, and two flows run daily to check for granted awards and deadline dates.
